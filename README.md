# 🚀 VocaSpark

VocaSpark is a modern vocabulary learning web application built with HTML, Tailwind CSS, DaisyUI, Vanilla JavaScript, and APIs 
to make English learning simple, interactive, and engaging. ✨📚🚀

## ✨ Features

- Learn new English words
- View meanings and definitions
- Listen to pronunciations
- Explore example sentences
- Responsive and user-friendly design

## 🛠️ Built With

- HTML5
- CSS3
- Tailwind CSS
- DaisyUI
- Vanilla JavaScript
- REST APIs
